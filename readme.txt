Post Gallery Facebook shortcodes for easily adding Gallery to your WordPress Theme, with a simple interface.

== Description ==

Post Gallery Facebook is an easy-to-use plugin that allows users to add Gallery into a Page or Post using the shortcode. Post Gallery Facebooks allows you to display a grid of images from the facebook page posts anywhere on your website. 
Create Facebook app and get the access tokens and page name and start using the gallery.


= Why Choose Post Gallery Facebook? =

= Post Gallery Facebooks is a great plugin for: =

* Adding gallery of facebook page posts.
* Limit the posts
* Fetch images from any page you want
* Displaying an Images in tile with animation.


= Pro Features include: =

* Facebook Posts Carousel: The Facebook Posts carousel shows 1 or more gallery on each slide. Your visitors can click Previous and Next buttons to scroll through the list of carousel.


= Simple To Use, but Deeply Customizeable =

Post Gallery Facebook is packed with options that are easy to use for anyone who is familiar with WordPress, but give you great control over the display of your Gallery.

= Add Images To Your Gallery To Stand Out = 

You'll be able to add an image to your Gallery, which can greatly increase the impact of your Gallery. You can share great social experience on your website.

= Supports Short Description for your Gallery = 

Post Gallery Facebook allows display of custom excerpted Gallery. Uses short description field to display excerpted Testimonial if the text get too lengthy.


= Why Do I Need Gallery on My Website? =

Gallery are a great thing to add to your website, for many reasons.

* Gallery help potential customers get to know that you are a credible business.
* Gallery give you the opportunity to point out specific features or compelling reasons why a customer should buy from you.
* Gallery, when used effectively, are a great tool to increase conversions rates on your website!

= Shortcode =

Use shortcode into a Page or Post
* [PDFacebookGallery]

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the contents of `/facebook-posts-gallery/` to the `/wp-content/plugins/` directory
2. Activate Post Gallery Facebook through the 'Plugins' menu in WordPress
